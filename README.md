# photo_frame
Add a frame to a photo

Example usage:
```shell
python3 -m add_border ~/Pictures/2025/20251122-Lausanne-Muensingen_Photo_Walk/Output/20251122-Lausanne-Muensingen_Photo_Walk-0116\ 1.jpg ~/Pictures/2025/20251122-Lausanne-Muensingen_Photo_Walk/Output/20251122-Lausanne-Muensingen_Photo_Walk-0116_instagram_border.jpg --width 1080 --height 1080 --pad-left 40 --pad-right 40 --bg-color white --quality 95
```
